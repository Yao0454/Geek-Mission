# HTML学习笔记

> HTML 是 **HyperText Markup Language** 的缩写，中文意思是“**超文本标记语言**”。
> HTML是网页的结构层，像是房子的**骨架和结构**。它定义了网页的基本组成部分，比如哪里是标题、哪里是段落、哪里是图片。

---
### 第一个HTML网页

```HTML
<!DOCTYPE html>
<html>
  <head>
    <title>我的第一个网页</title>
  </head>
  <body>
    <h1>Hello，HTML！</h1>
    <p>这是我的第一个网页，我正在学习 HTML。</p>
  </body>
</html>
```

- `<!DOCTYPE html>`: 这是一个文档类型声明，他会告诉浏览器，你读取的是一个HTML5规范的网页，通常这是每个HTML文件的第一行。
- `<html>`:这是整个HTML文档的根本元素，就好像函数的一个花括号，其他的所有元素都包含在里面。
- `<head>`:这是**头部**元素。他里面会包含一些**元数据**，就是描述网页自身信息的内容，这些内容通常不会直接显示给用户看。
	 - `<head>`:这是网页的标题，会显示在标签页上，而不是内容区
- `<body>`:这里是网页的主体部分。会显示你想让用户看到的所有内容，包括文字、图片、视频等。
	- `<h1>`:这是一级标题（Heading 1）。HTML中有`<h1>`到`<h6>`共六级标题，这个跟 Markdown 差不多。
	- `<p>`:这个是一个段落（Paragraph）。网页上的文本段落通常用段落包裹。

---
### HTML 的核心：标签 (Tags) 和元素 (Elements)

HTML 的基本构成单位是**元素 (Element)**。一个元素通常由一个**开始标签**、内容，和一个**结束标签**组成。

例如，定义一个段落：
```HTML
<p>这是一个段落。</p>
```

- `<p>` 是**开始标签** (Opening Tag)。
- `这是一个段落。` 是**内容** (Content)。
- `</p>` 是**结束标签** (Closing Tag)。

整个 `<p>这是一个段落。</p>` 就构成了一个完整的**段落元素**。

---
### 第一步：标题 (Headings) 和段落 (Paragraphs)

这是构建任何文本文档的基础。
#### 标题标签 `<h1>` 到 `<h6>`

在 HTML 中，我们有六个级别的标题标签，从 `<h1>` (最重要) 到 `<h6>` (最不重要)。它们用来定义文章或页面的标题和子标题。

- `<h1>`：通常用作页面的主标题，**一个页面最好只使用一次 `<h1>`**。这对于搜索引擎优化和网页可访问性很重要。
- `<h2>`：二级标题，用于划分主要部分。
- `<h3>` 到 `<h6>`：更低级别的子标题。

```HTML
<h1>这是一级标题 (文章主标题)</h1>
<h2>这是一个二级标题 (章节标题)</h2>
<p>这里是一些介绍章节内容的文字。</p>
<h3>这是一个三级标题 (小节标题)</h3>
<p>这里是一些更详细的文字。</p>
<h6>这是六级标题 (最次要的标题)</h6>
```

**注意**：
- 不要为了让文字变大或加粗而使用标题标签。标题有其结构上的意义。
- 按照层级顺序使用标题，不要跳级（比如在 `<h1>` 后面直接跟一个 `<h3>`）。

#### 段落标签 `<p>`

段落标签 `<p>` (Paragraph) 是你最常用的标签之一。任何时候你想开始一个新段落，都应该用 `<p>` 把它包起来。

浏览器会自动在段落元素的前后添加一些空白（外边距），从而将它们区分开。

```HTML
<!DOCTYPE html>
<html>
<head>
  <title>标题和段落练习</title>
</head>
<body>

  <h1>HTML 学习之旅</h1>
  <p>这是我学习 HTML 的第一天。我正在学习如何使用标题和段落标签来构建一个基本的网页结构。</p>

  <h2>什么是段落？</h2>
  <p>段落标签 <p> 用于定义一段文本。浏览器会自动在段落之间添加一些空间，让文章更容易阅读。</p>
  <p>这是另一个段落。注意，即使你在源代码中写了很多个空格或者换行，浏览器在显示时也只会把它们当作一个空格处理。只有使用新的 <p> 标签才能真正开始一个新段落。</p>

</body>
</html>
```

### 第二步：格式化文本 (Text Formatting)

有时候，我们不仅仅是想显示一段文本，还想让其中某些词语或句子突出显示，比如加粗或倾斜。HTML 提供了专门的标签来做这件事。

这里有一个非常重要的概念：**语义化 (Semantic)**。有些标签不仅改变样式，还赋予了文字特殊的含义。

#### 加粗文本：`<strong>` vs `<b>`

你想让一段文字变粗，有两个标签可以用：

1. `<strong>` (Strong Importance): 这个标签告诉浏览器，它包裹的文字是**非常重要**的、需要被强调的。浏览器默认会把它显示为粗体，但更重要的是，搜索引擎和屏幕阅读器（为视障人士服务）会知道这部分内容是重点。
2. `<b>` (Bold): 这个标签的作用很简单，就是让文字在视觉上**变粗**，它没有提供任何额外的语义信息。

**使用原则**：

- 当你想强调内容的**重要性、严肃性或紧急性**时，使用 `<strong>`。
- 如果只是为了让文字在视觉上吸引眼球，而没有特殊的重要性，可以使用 `<b>`。

```HTML
<p>在继续操作之前，请<strong>务必</strong>阅读用户协议。</p>
<p>这是一个<b>加粗</b>的单词，但它并不比其他词更重要。</p>
```

#### 倾斜文本：`<em>` vs `<i>`

同样，让文字倾斜也有两个选择：

1. `<em>` (Emphasis): 这个标签用来**强调**文本，改变一句话的重音。屏幕阅读器在读到 `<em>` 标签时，会用重读的语气来朗读它。浏览器默认将其显示为斜体。
2. `<i>` (Italic): 这个标签通常用来表示一段不同性质的文字，比如一个技术术语、一句外语、一个想法或者一本书的标题。它在视觉上是**斜体**，但没有强调的语义。

**使用原则**：

- 当你想**改变一句话的语调或重音**时，使用 `<em>`。
- 当你在引用书名、电影名、外语单词时，使用 `<i>`。

```HTML
<p>我<em>真的</em>很喜欢学习 HTML！</p>
<p>这本小说的名字是<i>《三体》</i>。</p>
```

### 第三步：创建列表 (Lists)

在网页上，我们经常需要把信息整理成列表的形式，这样看起来更清晰、更有条理。HTML 提供了两种主要的列表类型：无序列表和有序列表。这些东西和 Markdown 里面的都和相似。

无论哪种列表，列表中的每一个项目都使用同一个标签：`<li>` (List Item)。

#### 无序列表 `<ul>`

无序列表 (`<ul>` - Unordered List) 用于展示一组没有特定顺序的项目。浏览器通常会在每个项目前显示一个小圆点（项目符号）。

它非常适合用来创建：

- 购物清单
- 功能特性列表
- 导航菜单

```HTML
<p>我需要购买的物品：</p>
<ul>
  <li>牛奶</li>
  <li>面包</li>
  <li>鸡蛋</li>
</ul>
```

#### 有序列表 `<ol>`

有序列表 (`<ol>` - Ordered List) 用于展示一组有先后顺序的项目。浏览器会自动为每个项目添加数字编号（1, 2, 3...）。

它非常适合用来创建：

- 食谱步骤
- 操作指南
- 排行榜

```HTML
<p>早晨的例行公事：</p>
<ol>
  <li>起床</li>
  <li>刷牙洗脸</li>
  <li>吃早餐</li>
</ol>
```

#### 嵌套列表 (Nested Lists)

你还可以在一个列表项 `<li>` 内部再放入一个新的列表，从而创建出层级结构，就像写文档大纲一样。

```HTML
<p>课程大纲：</p>
<ul>
  <li>第一章：HTML 基础
    <ol>
      <li>标题和段落</li>
      <li>文本格式化</li>
      <li>列表</li>
    </ol>
  </li>
  <li>第二章：HTML 进阶</li>
</ul>
```

#### 下面是一个整合代码:
```HTML
<!DOCTYPE html>
<html>
<head>
  <title>列表练习</title>
</head>
<body>

  <h1>购物清单</h1>
  <p>这个周末去超市，<strong>不要</strong>忘记买下面的东西：</p>
  <ul>
    <li>蔬菜
      <ul>
        <li>西红柿</li>
        <li>西兰花</li>
      </ul>
    </li>
    <li>水果</li>
    <li>牛奶</li>
  </ul>

  <h1>制作三明治的步骤</h1>
  <p>按照<em>正确</em>的顺序来操作：</p>
  <ol>
    <li>准备两片面包</li>
    <li>在面包上涂上你喜欢的酱料</li>
    <li>加入生菜、火腿和奶酪</li>
    <li>把两片面包合在一起</li>
  </ol>

</body>
</html>
```

### 第四步：添加链接 (Links)

链接是“超文本”的核心，它允许我们从一个页面跳转到另一个页面，将整个网络连接起来。

创建链接的标签是 `<a>`，它代表 **Anchor (锚)**。

一个最基本的链接由两部分组成：

1. `<a>` 标签本身，以及最重要的属性：`href`。
2. 你想让用户点击的**文本内容**，它被放在 `<a>` 和 `</a>` 标签之间。

`href` 属性 (Hypertext Reference) 用来指定链接的目标地址，也就是你希望用户点击后跳转到的地方。

#### 1. 链接到外部网站（绝对路径）

如果你想链接到一个完全不同的网站，你需要提供完整的 URL 地址，包括 `https://`。

**示例代码：**

```HTML
<p>你可以使用 <a href="https://www.google.com">Google</a> 来搜索信息。</p>
<p>学习更多编程知识，可以访问 <a href="https://github.com">GitHub</a>。</p>
```

#### 2. 链接到自己网站的其他页面（相对路径）

假设你的网站有多个页面，比如 `index.html` (首页) 和 `about.html` (关于我们)。你可以在它们之间建立链接。如果这两个文件在同一个文件夹里，你只需要直接使用文件名。

**示例代码 (`index.html`)：**

```HTML
<p>欢迎来到我的主页！点击这里了解<a href="about.html">关于我的更多信息</a>。</p>
```

**对应的 `about.html` 文件：**

```HTML
<!DOCTYPE html>
<html>
<head>
  <title>关于我</title>
</head>
<body>
  <h1>关于我</h1>
  <p>这是一个关于我的页面。</p>
  <p>点击<a href="index.html">返回首页</a>。</p>
</body>
</html>
```

#### 3. 在新标签页中打开链接

默认情况下，链接会在当前浏览器标签页中打开。如果你希望用户点击链接后，在一个**新的标签页**中打开目标网页（这样用户就不会离开你的网站），你可以添加 `target="_blank"` 属性。

```HTML
<p>访问 <a href="https://www.wikipedia.org/" target="_blank">维基百科</a> 获取知识 (将在新标签页打开)。</p>
```

#### 下面是两个完整示例文件：
- page1.html:
```HTML
<!DOCTYPE html>
<html>
<head>
  <title>第一页</title>
</head>
<body>

  <h1>第一页</h1>
  <p>这是我的第一个页面。</p>

  <p>我想从这里跳转到 <a href="page2.html">第二页</a>。</p>

</body>
</html>
```

- page2.html:
```HTML
<!DOCTYPE html>
<html>
<head>
  <title>第二页</title>
</head>
<body>

  <h1>欢迎来到第二页！</h1>
  <p>你成功地从第一页跳转过来了。</p>

  <p>现在，让我们<a href="page1.html">返回第一页</a>。</p>

</body>
</html>
```

### 第五步：插入图片 (Images)

要在网页上显示图片，我们使用 `<img>` 标签。

`<img>` 标签是一个**空标签**（或自闭合标签），这意味着它没有结束标签 `</img>`。它通过**属性**来工作。

最重要的两个属性是 `src` 和 `alt`。

#### 1. `src` 属性 (Source)

`src` 属性告诉浏览器图片的**来源地址**。这个地址可以是网络上的一个链接，也可以是你电脑上的一个本地文件路径。
- **链接网络图片（绝对路径）**： 你可以直接使用网上的图片链接。这是最简单的方式，因为你不需要自己准备图片文件。
```HTML
<img src="https://www.w3.org/html/logo/downloads/HTML5_Logo_256.png">
```
- **链接本地图片（相对路径）**： 在实际项目中，我们通常会把图片文件和 HTML 文件放在一起。一个好的习惯是创建一个 `images` 文件夹来存放所有图片。

假设你的项目结构是这样的：
```
my-website/
    ├── index.html
    └── images/
        └── cat.jpg
```
那么在 `index.html` 中，你可以这样引用 `cat.jpg`：
```HTML
<img src="images/cat.jpg">
```

#### 2. `alt` 属性 (Alternative Text)

`alt` 属性为图片提供一段**替代文本**。这个属性**极其重要**，原因有三：

1. **可访问性 (Accessibility)**：对于视障用户，屏幕阅读器会朗读 `alt` 属性中的文字，让他们知道图片的内容。
2. **容错性**：如果图片因为网络问题或路径错误而无法加载，浏览器会显示 `alt` 属性的文本。
3. **SEO (搜索引擎优化)**：搜索引擎会读取 `alt` 文本来理解图片内容，有助于提高网站排名。

**Gemini告诉我永远不要忘记给你的图片添加 `alt` 属性！**

```HTML
<img src="images/a-cute-orange-cat.jpg" alt="一只可爱的橘猫正躺在草地上晒太阳">
```

如果图片纯粹是装饰性的，没有任何实际内容，`alt` 属性也应该保留，但可以设置为空：`alt=""`。

#### 3. `width` 和 `height` 属性

你可以使用 `width` (宽度) 和 `height` (高度) 属性来指定图片在页面上显示的尺寸。属性值是像素数。
```HTML
<img src="https://www.w3.org/html/logo/downloads/HTML5_Logo_256.png" alt="HTML5 Logo" width="128" height="128">
```

**Tips:**：

- 只设置 `width` 或 `height` 中的一个，浏览器会自动按比例缩放另一个，可以防止图片变形。
- 在现代网页设计中，我们更多地使用 CSS 来控制图片尺寸以实现响应式设计，但 `width` 和 `height` 属性依然很有用，因为它可以帮助浏览器在图片加载前预留出正确的空间，防止页面布局跳动。

#### 下面是一个完整示例：
```HTML
<!DOCTYPE html>
<html>
<head>
  <title>图片练习</title>
</head>
<body>

  <h1>我最喜欢的动物</h1>
  <p>下面是一张非常可爱的猫咪图片。</p>

  <img src="https://placekitten.com/400/300" alt="一只可爱的小猫，正好奇地看着镜头" width="400">

  <h2>图片链接</h2>
  <p>你甚至可以把图片放在链接标签 <a> 里面，让图片变成一个可以点击的链接！</p>
  
  <a href="https://www.wikipedia.org/wiki/Cat" target="_blank" rel="noopener noreferrer">
    <img src="https://placekitten.com/200/287" alt="一只小猫的肖像照" width="200">
  </a>
  <p>(点击上面的小猫图片，了解更多关于猫的知识)</p>

</body>
</html>
```
