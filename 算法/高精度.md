# 高精度运算

> - 很多时候在除了很大数据流的加法以及乘法时，使用传统运算符所带来的误差是巨大的，高精度运算就是解决他的一个高效的方法。

- 高精度运算的本质其实就是利用字符串来针对每一位进行计算。

## 1. 高精度加法

```C++
#include <bits/stdc++.h>
using namespace std;

string add_num(string num1, string num2) {
	string result = "";
	int carry = 0;
	
	int i = num1.length() - 1;
	int j = num2.length() - 1;
	
	//加法从低位往高位算
	
	while(i >= 0 || j >= 0 || carry) {
	//只有当两数的所有位数都运算完，以及进位也计算完，才停止循环
		int sum = carry;
		
		if (i >= 0) sum += num1[i--] - '0'; //一种更简洁的写法
		/*
		等同于
		if (i >= 0) {
			sum += num1[i] - '0';
			i--;
		}
		*/
		if (j >= 0) sum += num2[j--] - '0';
		
		result = char(sum % 10 + '0') + result;
		carry = sum / 10;
	}
	
	return result;
}
```

## 2. 高精度乘法
```C++
#include <bits/stdc++.h>
using namespace std;

string multiply(string num, int x) {
string result = "";
int carry = 0;

for (int i = num.length() - 1; i >= 0; i--) {
	int temp = (num[i] - '0') * x + carry;
	
	result = char(temp % 10 + '0') + result;
	carry = temp / 10;
}

while (carry) {
	result = char(carry % 10 + '0') + result;
	carry /= 10;
}
return result;
}
```
